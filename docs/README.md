# Docs Deployment (GitHub Pages)

此專案文件放在 `docs/`，可直接用 GitHub Pages 佈署。

## 方式一：GitHub Pages（建議）

1. 到 GitHub repository settings
2. 開啟 `Pages`
3. `Build and deployment` 選 `Deploy from a branch`
4. Branch 選 `main`（或你的預設分支）
5. Folder 選 `/docs`

完成後 Pages 會把 `docs/` 內容作為文件站來源。

## 文件入口

- `index.md`
- `getting-started.md`
- `cli.md`
- `sdk.md`
